Каталог видеоигр Nintendo
Каталог видеоигр для домашних консолей Nintendo — мобильное приложение под Android на Java. Полноценный CRUD-каталог с поиском, фильтрами по платформе/жанру, избранным с рейтингом/комментарием, оффлайн-доступом и стилизацией в фирменном стиле Nintendo (красный Toolbar, белый фон).

Функционал

Каталог игр — список с обложками, названием, годом, иконкой платформы (NES, SNES, Switch), жанром.
Поиск по названию (SearchView в Toolbar).
Фильтры по платформе (NES, SNES, N64, GameCube, Wii, Wii U, Switch) и жанру (платформер, RPG, шутер и др.).
Сортировка (по названию, году, платформе).
Детали игры — большая обложка, описание, платформа, жанр, разработчик, рейтинг, комментарий.
Избранное — отдельный экран с рейтингом, редактированием/удалением.
CRUD — добавление, редактирование/удаление (long click на карточку).
Оффлайн — Room кэширует данные, избранное/рейтинг/комментарий сохраняются локально.
Дизайн — Nintendo-стиль: красный Toolbar/статус-бар, белые иконки, минималистичные карточки.

Технологии

Java + Android SDK (minSdk 21).
MVVM + LiveData + ViewModel.
Retrofit + OkHttp + Gson (mock-сервер).
Room (SQLite с оффлайн-хранением).
Glide (загрузка обложек/иконок платформ).
RecyclerView + Material Components (Toolbar, CardView, FAB, DialogFragment).
CoordinatorLayout (правильный отступ под Toolbar).

Архитектура: Model (Game) → Repository (слияние сеть+Room) → ViewModel → UI (Activity + Adapter).
